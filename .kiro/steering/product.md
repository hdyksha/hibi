# Product Overview

## Todo App

A simple task management application built with a modern full-stack architecture.

### Key Features
- Task creation, editing, and deletion
- Task filtering and archiving
- Markdown support for task descriptions
- Real-time network status monitoring
- Responsive design for mobile and desktop

### Architecture
- **Frontend**: React-based SPA with TypeScript
- **Backend**: RESTful API server with Express.js
- **Data Storage**: JSON file-based persistence
- **Development**: Hot-reload development servers with concurrent execution

### Target Users
Users who need a lightweight, self-hosted task management solution with markdown support and offline capabilities.
